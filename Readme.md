# Portfolio – Aïna Développeuse Web

## Description

Ce projet correspond à la création de mon portfolio de développeuse web réalisé dans le cadre du Projet 8 de la formation OpenClassrooms.

Ce portfolio présente mon profil, mon parcours, mes compétences, ainsi que plusieurs projets réalisés durant la formation.
Le site a été conçu avec une attention particulière portée à l’accessibilité, au SEO et au responsive design.

## Démo en ligne

https://biou007.github.io/Projet-8/
